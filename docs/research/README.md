# Research Reports

This directory stores generated and external research artifacts that may inform product or architecture decisions, but are not themselves the current source of truth.

Use these reports as supporting context only. For active product state, use `STATUS.md`, `CONTEXT.md`, `PRD.md`, `ROADMAP.md`, `docs/shipped-vs-planned.md`, and the latest ADRs first.
