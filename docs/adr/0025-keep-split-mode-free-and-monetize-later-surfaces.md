# Keep Split Mode free and monetize later surfaces

Compass monetization research supports the existing product decision that normal Split Mode tracking and USDC Settlements stay free at launch. FundWise should monetize later surfaces instead: invite-only Fund Mode, Fundy premium / agent workflows, paid agent endpoints, and partner rails once those are actually implemented. We will not launch a token or add a normal Settlement fee for the hackathon submission or first Split Mode launch because that would weaken the viral Group loop before the product has durable usage.
