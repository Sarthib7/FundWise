# Fix USDC as the MVP settlement asset

FundWise originally allowed broader stablecoin flexibility, but the current MVP is optimized for one-click Split Mode settlement on Solana mainnet-beta. To keep Group Balances, Settlement Request Links, Receipts, insufficient-funds handling, and LI.FI top-ups simple and consistent, the MVP fixes the settlement asset to USDC. This supersedes the broader per-Group mint flexibility described earlier for MVP scope.
