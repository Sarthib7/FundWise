# Squads governance is the Fund Mode authority layer

Fund Mode is the sovereign paid product surface, so Treasury authority must live in Squads rather than in FundWise database rows. FundWise may store Proposal metadata, proof, cached status, and mappings to Squads transaction/proposal accounts, but Proposal creation, approvals, rejections, and execution are wallet-confirmed Squads actions; database records only mirror on-chain governance state for UX and history.
